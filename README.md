# IT323 - MIDTERM PIT
## System Project: Student Record Management System
A **full-stack web application** for managing student records with CRUD operations, search functionality, and data visualization. Built with Node.js, MySQL, and vanilla JavaScript.

## Technology Used
### Backend
- **Node.js** – JavaScript runtime environment.
- **Express.js** – Web framework for Node.js.
- **MySQL** – Relational database management.
- **CORS** – Cross-Origin Resource Sharing middleware.

### Frontend
- **HTML5 & CSS3** – Semantic markup and modern styling (Flexbox/Grid).
- **JavaScript (ES6+)** – Vanilla JS for DOM manipulation and API calls.
- **Chart.js** – Library for data visualization.
- **Google Fonts** – DM Sans and DM Mono typography.
