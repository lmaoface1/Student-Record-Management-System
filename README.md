# IT323 - MIDTERM PIT
## System Project: Student Record Management System
A **full-stack web application** for managing student records with CRUD operations, search functionality, and data visualization. Built with Node.js, MySQL, and vanilla JavaScript.

## Features
- Complete CRUD Operations: Create, Read, Update, and Delete student records with ease.
- Search & Filter: Instantly find students by Name or ID and filter results by specific courses.
- Data Visualization: Interactive charts (via Chart.js) displaying student distribution by course and year level.
- Responsive Design: A modern, "mobile-first" UI that scales perfectly from smartphones to desktops.
- Dark/Light Mode: User-selectable themes for a personalized viewing experience.
- Sorting & Pagination: Organize large datasets and navigate through records without lag.

## Technology Used
### Backend
- **Node.js** – JavaScript runtime environment.
- **Express.js** – Web framework for Node.js.
- **MySQL** – Relational database management.
- **CORS** – Cross-Origin Resource Sharing middleware.

### Frontend
- **HTML5 & CSS3** – Semantic markup and modern styling (Flexbox/Grid).
- **JavaScript (ES6+)** – Vanilla JS for DOM manipulation and API calls.
- **Chart.js** – Library for data visualization.
- **Google Fonts** – DM Sans and DM Mono typography.
